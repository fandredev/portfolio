## Clone this repo

###### Notes: Please, check the .tool-versions before use this project to show versions that i used.

```
git clone git@github.com:fandredev/portfolio.git
```

## Go to directory

```
 cd portfolio
```

## Install dependencies

```
 yarn
```

## Run server

```
 yarn run dev
```

## Run tests

```
 yarn run test
```

## Run coverage

```
 yarn run coverage
```

## Run linter

```
 yarn run lint
```
