## Clone this repo

```
git clone git@github.com:fandredev/portfolio.git
```

## Go to directory

```
 cd portfolio
```

## Install dependencies

```
 yarn
```

## Run server

```
 yarn run dev
```

## Run tests

```
 yarn run test
```

## Run coverage

```
 yarn run coverage
```

## Run linter

```
 yarn run lint
```

## Run UI components

```
 yarn run storybook
```
